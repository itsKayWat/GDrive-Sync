GDrive_One-Backup

Why This Tool?

GDrive_One-Backup was developed to provide a simple, visual solution for maintaining Google Drive backups. It eliminates the complexity of manual backups while providing real-time progress tracking and folder management.

Features

  • Visual backup progress tracking
  • Multiple folder management
  • Secure Google OAuth2 authentication
  • Dark theme interface
  • Resumable file uploads

Prerequisites

• Python 3.7+ • Google Cloud Console Account • Google Drive API enabled • OAuth 2.0 credentials • Required Python packages

Installation Guide

  1. Download/clone the repository
  2. Install requirements:
    pip install -r requirements.txt
  3. Place credentials.json in root directory
  4. Run launcher.pyw

Prevention Scenarios

Hardware Failure Protection

Regular backups ensure data survival in case of drive failures

Ransomware Protection

Separate cloud storage prevents complete data loss from ransomware attacks

Accidental Deletion

Cloud copies enable easy file recovery

Important Notes

  • Keep credentials.json secure
  • Regular backups recommended
  • Monitor Google Drive storage limits

Quick Start

1. Run launcher.pyw 2. Click "Login to Google" 3. Add folders to backup 4. Click "Sync Now"
